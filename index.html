Expert Validator Help Guide

Welcome to the Chemical RTD Extraction Validation Tool. This website helps non-technical experts validate and correct structured chemical research paper metadata.

How to Validate Fields:

  • Edit values directly in the input boxes. Long text fields automatically expand.
  • Arrays / Lists are rendered as capsules/pills. Click the × on a pill to delete it. To add a value, click inside the input box, type the value, and press Enter.
  • Nested Lists (Objects): You can add new objects or delete entries by using the dedicated buttons.
  • Tables (Spreadsheet Grid): For table rows, double-click/focus any cell, edit it like Excel, and add or delete rows as needed.
  • LaTeX Equations: Dynamic preview of LaTeX formulas will render automatically when editing equations.

Field-Level Flagging:

If a specific value is suspicious or incorrect and you're unsure how to fix it, click the flag icon (🏴) next to that field. It will highlight in orange and save the path in the exported data for further review.

Keyboard Shortcuts:

Ctrl + S Validate current document, save changes, and advance to next Ctrl + Right Arrow Advance to next document in list Ctrl + Left Arrow Go back to previous document in list Ctrl + F Toggle whole document status flag Esc Close this help dialog
